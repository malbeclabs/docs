# 欢迎来到 DoubleZero

DoubleZero 是一种高性能去中心化协议和网络，专为区块链等分布式系统优化。DoubleZero 的用户可享受高度优化的分布式系统数据传输和预处理的集成服务。

## DoubleZero 面向哪些群体？

### 贡献者

DoubleZero 贡献者拥有未充分利用的私有光纤链路和硬件（例如交换机和 FPGA 设备）形式的网络容量。将这些资源配置为扩展 DoubleZero 网络，可以创建一个集成的信息服务。通过 DZ 协议，网络贡献者将这些网络容量整合为一个动态且广泛的全球网状网络，并因其贡献获得代币激励。

### 用户

DoubleZero 用户参与需要高性能网络连接的分布式系统，例如区块链。通过将区块链系统与 DoubleZero 互连，用户可以获得一系列优化功能来加速其网络 I/O。

在初始实现中，DoubleZero 网络兼容区块链验证者客户端以及 RPC 和 MEV 基础设施提供商。这些用户系统在获得网络访问权限之前，需要通过在各自账本上验证其唯一标识的公钥地址来进行准入控制。随着网络逐步纳入来自其他生态系统的用户系统，将针对每个用户项目社区添加额外的身份识别和认证策略。

## 优势

DoubleZero 的唯一替代方案是公共互联网。与公共互联网相比，DoubleZero 网络为区块链提供了两项改进。

*第一*，入站交易可以在网络贡献者硬件层面以非歧视性的方式进行边缘过滤（即去除垃圾信息和重复数据），然后再发送给 DoubleZero 网络的用户（例如区块链节点）。这使得区块链能够受益于共享的全系统过滤资源，而无需每个验证者单独配置足够的资源。因此，验证者可以将其资源重新分配到直接产生收益的活动中。此功能目前尚未上线。


*第二*，出站消息的路由更加直接且经过优先级排序以提高效率。这是因为在 DoubleZero 中，全球数据传输的路由逻辑针对速度进行了优化，与公共互联网不同。因此，验证者受益于更低的抖动——抖动是指两个数据线缆端点之间延迟变化的幅度。本质上，区块链节点可以更快地发送和接收区块。

综合来看，DoubleZero 网络的这些优势可以实现通过公共互联网无法达到的性能水平。

如需了解更多详情，请参阅 [DoubleZero 协议](https://doublezero.xyz/whitepaper.pdf)白皮书。
