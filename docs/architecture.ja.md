## コントリビューター

DoubleZeroネットワークは、世界中の都市に分散するネットワークインフラプロバイダーの成長するコミュニティによる接続性とパケット処理の貢献によって構成されています。コントリビューターは、分散型メッシュネットワークを提供するために、光ファイバーケーブルリンクと情報処理リソースをプロトコルに提供します。

### ネットワーク帯域幅コントリビューター

ネットワークコントリビューターは、2つのポイント間の専用帯域幅を提供し、各端でDoubleZero互換デバイス（DZD）を運用し、各端でインターネット接続を確保する必要があります。また、ネットワークコントリビューターは、マルチキャスト、ユーザールックアップ、エッジフィルタリングサービスなどのサービスを提供するために、各DZDでDoubleZeroソフトウェアを実行する必要があります。

DoubleZeroネットワークの物理リンクは、一般にウェーブレングスサービスと呼ばれる光ファイバーケーブルの形で提供されます。ネットワークコントリビューターは、2つ以上のデータセンター間で、インフラプロバイダーから所有またはリースした未活用のネットワークリンクを提供します。これらのリンクは、DoubleZero Agentソフトウェアのインスタンスを実行する物理的なネットワークスイッチングエンクロージャであるDoubleZero Deviceによって両端で終端されます。

#### DoubleZero Exchange（DZX / クロスコネクトサイト）

DoubleZero Exchange（DZX）は、異なるコントリビューターのリンクがブリッジされるメッシュネットワークの相互接続ポイントです。DZXは、ネットワークの交差点が存在する世界中の主要な大都市圏に設置されています。ネットワークコントリビューターは、リンクのエンドポイントに地理的に最も近いDZXで、自身のリンクをより広範なDoubleZeroメッシュネットワークにクロスコネクトする必要があります。

### 計算リソースコントリビューター

ネットワークコントリビューターとは別に、リソースコントリビューターは、DoubleZeroネットワークの技術的整合性と継続的な機能性を維持するために必要なさまざまな保守および監視業務を行う分散型のネットワーク参加者グループです。具体的には、(i) ユーザートランザクションと支払いの追跡、(ii) ネットワークコントリビューターの手数料計算、(iii) (i)および(ii)の結果の記録、(iv) プロトコルのトークノミクスを制御するスマートコントラクトの非裁量的な管理、(v) 該当するブロックチェーンへのアテステーションのリレー、(vi) すべてのネットワークコントリビューターに対して透明性のあるリアルタイムのパフォーマンスメトリクスを提供するためのリンク品質と利用率に関するテレメトリデータの公開を行います。

## コンポーネント

### DoubleZero Daemon

DoubleZero Daemonソフトウェアは、DoubleZeroネットワークを介して通信する必要があるサーバー上で実行されます。デーモンは、ホストのカーネルネットワーキングスタックとインターフェースし、トンネルインターフェース、ルーティングテーブル、およびルートの作成と管理を行います。

### Activator

Activatorサービスは、DoubleZeroコミュニティの1つ以上の計算リソースコントリビューティングメンバーによってホストされ、IPアドレスの割り当てと状態変更を必要とするコントラクトイベントを監視し、ネットワークに代わってそれらの変更を管理します。

### Controller

Controllerサービスは、DoubleZeroコミュニティの1つ以上の計算リソースコントリビューターによってホストされ、スマートコントラクトイベントに基づいてDoubleZero Device Agentが現在の構成をレンダリングするための構成インターフェースとして機能します。

### Agent

AgentソフトウェアはDoubleZero Device上で直接実行され、Controllerサービスによって解釈された構成変更をデバイスに適用します。AgentソフトウェアはControllerに構成変更をポーリングし、オンチェーンの正規のデバイス状態とデバイス上のアクティブな構成の差分を計算し、アクティブな構成を整合させるために必要な変更を適用します。

### Device

DoubleZeroネットワークのルーティングとリンク終端を提供する物理デバイスエンクロージャです。DZDはDoubleZero Agentソフトウェアを実行し、Controllerサービスから読み取ったデータに基づいて構成されます。
