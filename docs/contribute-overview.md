# Contributor Documentation

Welcome to the DoubleZero contributor documentation. This section covers everything you need to become a network contributor.

## Prerequisites

Before provisioning devices, complete the [Setup](setup.md) guide to install the DoubleZero CLI.

## Documentation Structure

| Guide | Description |
|-------|-------------|
| [Requirements & Architecture](contribute.md) | Contributor requirements, network architecture concepts, bandwidth options, and hardware specifications |
| [Device Provisioning](contribute-provisioning.md) | Complete A-Z guide to provision your first DZD - from key generation to agent installation |
| [Operations](contribute-operations.md) | Ongoing operational tasks: agent upgrades and monitoring |

## Getting Started

1. **Review requirements** - Understand what's needed to become a contributor
2. **Plan your hardware** - Review hardware specifications and data center requirements
3. **Provision your device** - Follow the step-by-step provisioning guide
4. **Operate** - Use the operations guide for ongoing maintenance

Ready to begin? Start with [Requirements & Architecture](contribute.md).
