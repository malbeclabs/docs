# User Documentation

Welcome to the DoubleZero user documentation. This section covers everything you need to connect to and use the DoubleZero network.

## Prerequisites

Before connecting to DoubleZero, complete the [Setup](setup.md) guide to install the DoubleZero CLI.

## Getting Started

1. **Install the CLI** - Follow the [Setup](setup.md) guide
2. **Choose your network** - Mainnet-Beta or Testnet
3. **Connect** - Follow the appropriate connection guide
4. **Pay fees** - Set up fee payments for continued access
