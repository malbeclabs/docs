## 기여자

DoubleZero 네트워크는 전 세계 도시에 분산된 네트워크 인프라 제공업체들의 성장하는 커뮤니티로부터 연결성과 패킷 처리 기여를 통해 구성됩니다. 기여자들은 분산형 메시 네트워크를 제공하기 위해 광섬유 케이블 링크와 정보 처리 자원을 프로토콜에 제공합니다.

### 네트워크 대역폭 기여자

네트워크 기여자는 두 지점 간에 전용 대역폭을 제공하고, 양쪽 끝에서 DoubleZero 호환 장치(DZD)를 운영하며, 양쪽 끝에서 인터넷 연결을 제공해야 합니다. 네트워크 기여자는 또한 각 DZD에서 DoubleZero 소프트웨어를 실행하여 멀티캐스트, 사용자 조회, 엣지 필터링 서비스와 같은 서비스를 제공해야 합니다.

DoubleZero 네트워크의 물리적 링크는 일반적으로 파장 서비스라고 불리는 광섬유 케이블 형태로 제공됩니다. 네트워크 기여자는 두 개 이상의 데이터 센터 간에 인프라 제공업체로부터 소유하거나 임대한 활용도가 낮은 네트워크 링크를 할당합니다. 이 링크들은 양쪽 끝에서 DoubleZero Agent 소프트웨어 인스턴스를 실행하는 물리적 네트워크 스위칭 인클로저인 DoubleZero Device에 의해 종단됩니다.

#### DoubleZero Exchange (DZX / 상호접속 사이트)

DoubleZero Exchange(DZX)는 서로 다른 기여자 링크가 연결되는 메시 네트워크의 상호접속 지점입니다. DZX는 네트워크 교차점이 발생하는 전 세계 주요 대도시 지역에 위치합니다. 네트워크 기여자는 링크 엔드포인트에 지리적으로 가장 가까운 DZX에서 자신의 링크를 더 넓은 DoubleZero 메시 네트워크에 상호접속해야 합니다.

### 컴퓨팅 자원 기여자

네트워크 기여자와 별도로, 자원 기여자는 DoubleZero 네트워크의 기술적 무결성과 지속적인 기능을 유지하는 데 필요한 다양한 유지보수 및 모니터링 업무를 수행하는 분산된 네트워크 참여자 그룹입니다. 구체적으로, 이들은 (i) 사용자 트랜잭션 및 결제를 추적하고; (ii) 네트워크 기여자에 대한 수수료를 계산하며; (iii) (i)과 (ii)의 결과를 기록하고; (iv) 프로토콜 토크노믹스를 제어하는 스마트 컨트랙트를 엄격히 비재량적 기준으로 관리하며; (v) 해당 블록체인에 증명을 중계하고; (vi) 모든 네트워크 기여자에 대한 투명하고 실시간 성능 지표를 제공하기 위해 링크 품질 및 활용도에 관한 텔레메트리 데이터를 게시합니다.

## 구성 요소

### DoubleZero Daemon

DoubleZero Daemon 소프트웨어는 DoubleZero 네트워크를 통해 통신해야 하는 서버에서 실행됩니다. 데몬은 호스트의 커널 네트워킹 스택과 인터페이스하여 터널 인터페이스, 라우팅 테이블 및 경로를 생성하고 관리합니다.

### Activator

Activator 서비스는 DoubleZero 커뮤니티의 하나 이상의 컴퓨팅 자원 기여 멤버가 호스팅하며, IP 주소 할당 및 상태 변경이 필요한 컨트랙트 이벤트를 모니터링하고 네트워크를 대신하여 해당 변경 사항을 관리합니다.

### Controller

Controller 서비스는 DoubleZero 커뮤니티의 하나 이상의 컴퓨팅 자원 기여자가 호스팅하며, 스마트 컨트랙트 이벤트를 기반으로 현재 구성을 렌더링하기 위한 DoubleZero Device Agent의 구성 인터페이스 역할을 합니다.

### Agent

Agent 소프트웨어는 DoubleZero Device에서 직접 실행되며, Controller 서비스에 의해 해석된 구성 변경 사항을 장치에 적용합니다. Agent 소프트웨어는 Controller에 구성 변경 사항을 폴링하고, 온체인에 기록된 정식 Device 상태 버전과 장치의 활성 구성 간의 차이를 계산하며, 활성 구성을 일치시키기 위해 필요한 변경 사항을 적용합니다.

### Device

DoubleZero 네트워크를 위한 라우팅 및 링크 종단을 제공하는 물리적 장치 인클로저입니다. DZD는 DoubleZero Agent 소프트웨어를 실행하며, Controller 서비스에서 읽은 데이터를 기반으로 구성됩니다.
