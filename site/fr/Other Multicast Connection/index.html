    Informations de connexion détaillées :

    1. Installation du Client DoubleZero

    Veuillez suivre les instructions de configuration pour installer et configurer le client DoubleZero.

    2. Instructions de Connexion

    Connectez-vous à DoubleZero en mode Multicast En tant qu'éditeur :

    doublezero connect multicast --publish <nom du flux>

    ou en tant qu'abonné :

    doublezero connect multicast --subscribe <nom du flux>

    ou pour publier et s'abonner :

    doublezero connect multicast --publish <nom du flux> --subscribe <nom du flux>

    Pour publier ou s'abonner à plusieurs flux, vous pouvez inclure plusieurs noms de flux séparés par des espaces. Cela peut également être utilisé pour publier et s'abonner à des flux de publication. Par exemple doublezero connect multicast --subscribe feed1 feed2 feed3

    3. Vérifiez votre connexion multicast active.

    Attendez 60 secondes puis exécutez

    doublezero status
    Résultat attendu : - Session BGP active sur le bon réseau DoubleZero - Si vous êtes un éditeur, votre IP DoubleZero sera différente de votre IP Source Tunnel. C'est normal. - Si vous êtes uniquement abonné, votre IP DoubleZero sera identique à votre IP Source Tunnel.
